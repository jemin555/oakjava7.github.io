Welcome to OAKJAVA7 Programming Language Samples using Scroll Framework !

SCROLL is a java/j2ee framework invented in java-jdk1.8 by wilmix jemin for webapplication at year 2017.

SYNTAX: 



 < SCROLL >  

public class classname {

 static void main(String args[]) {



}

}

 < /SCROLL  >

ADVANTAGES:
  • USED only with Latest JAVA version (OAK JAVA) JAVA7.0
  • It is used to publish java web application like struts,spring ,hibernate, and JSF in mobiles.
  • IT has attractive syntax.
  • Developer need to concenterate only on core java so it is learnable. in other framework developer had to learn more syntax.
  • It cannot directly interact with Struts ,JSF but can publish the url.
  • It is used with JQUERY, bootstrap, Spring, Hibernate, mybatics , since it has main function.
